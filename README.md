# 程式應用與 AI 學習實戰 (Programming Applications and AI)

本專案提供了一套完整且循序漸進的 Python 程式設計與人工智慧（AI）學習資源。從基礎的資料結構到進階的機器學習實戰，旨在幫助學習者在最短的時間內掌握大數據分析與 AI 的核心技能。

## 📖 簡介

在現今時代，大數據分析和人工智慧的重要性不言而喻。無論是科技巨頭的全力投入，或是各行各業的數位轉型，AI 已成為不可或缺的競爭力。

本課程選擇 **Python** 作為入門工具，因為其易學易用且擁有強大的數據分析與機器學習套件（如 Pandas, Scikit-learn 等），能讓初學者在最短時間內建立模型並看到成效。

## 🎯 學習心法

*   **「兵貴速，不貴久」**：學習新技能需要專注且快速取得成效，以維持動機與意志力。
*   **「最短、最快」成功法則**：專注於核心任務，以壓倒性的專注力取得成果。

## 🗺️ 課程單元安排

本課程分為六大核心單元，涵蓋從基礎到實戰的全過程：

### 一、程式設計與 AI 基本概念
*   程式設計與 AI 簡史
*   認知科學與行為科學簡介
*   為何選擇 Python
*   知識管理工具介紹

### 二、Python 資料科學基礎
*   Google Colab 環境簡介
*   基礎資料結構：List, Dictionary, Array, DataFrame
*   套件、模組與函數
*   Numpy 與 Pandas 入門
*   流程控制：if, for, list comprehension

### 三、資料處理實務
*   資料取得：從 GitHub 與網址讀取資料
*   Pandas 資料讀取與清理
*   時序資料處理 (Time Series)

### 四、資料視覺化：股價與特徵分析
*   Pandas Plot：繪製股價走勢圖
*   資料分佈分析：Bar chart, Boxplot
*   Seaborn 進階視覺化

### 五、機器學習原理：鳶尾花分類
*   監督式與非監督式學習
*   特徵值 (Features) 的概念
*   Seaborn 套件應用
*   決策邊界 (Decision Region) 視覺化

### 六、實戰演練：鐵達尼號生存預測
*   鐵達尼號資料集分析
*   資料整理與缺值補全
*   資料切片與特徵工程
*   建立機器學習模型

### 🚀 進階與選修主題
*   **股價走勢預測**：Deep Learning, Keras, Neural Network, FB Prophet
*   **人臉偵測**：OpenCV, Haar 特徵
*   **自然語言處理 (NLP)**：gTTS 套件, 語音轉文字, 語意分析

## 📂 專案結構說明

本倉庫依單元組織，方便循序漸進學習：

*   `單元1/` 至 `單元6/`：存放各單元的 Jupyter Notebook (`.ipynb`) 與相關 Markdown 文件。
*   `data/`：存放課程所需的資料集。
*   `*.ipynb`：位於根目錄的各類實戰練習筆記本，包含 Plotly 視覺化、鐵達尼號模型等。
*   `*.PNG`：課程相關的概念圖示。

## 🚀 如何開始

1.  **環境準備**：建議使用 [Google Colab](https://colab.research.google.com/) 開啟本專案中的 `.ipynb` 檔案，無需安裝環境即可執行。
2.  **學習路徑**：建議依照 `單元1` 到 `單元6` 的順序進行練習。
3.  **實作練習**：每個單元都附有實作筆記本，請親自動手修改程式碼以加深理解。

## 🛠️ 使用工具與套件

*   **語言**：Python 3
*   **資料處理**：Pandas, Numpy
*   **視覺化**：Matplotlib, Seaborn, Plotly
*   **機器學習**：Scikit-learn, Keras, TensorFlow
*   **其他**：FB Prophet (時序預測), OpenCV (電腦視覺)
